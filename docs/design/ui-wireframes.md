## フィルタの挙動

- **都市**: タップで ON / OFF（複数選択、OR 条件）
- **ジャンル**: タップで ON / OFF（複数選択、OR 条件）
- **都市 × ジャンル**: AND 条件で組み合わせ
- **訪問済みのみ / 未訪問のみ**: 排他トグル
- **リセット**: すべてのフィルタを解除し、全店舗表示に戻す

フィルタの状態は URL のクエリパラメータに同期させ、共有可能にする
例: `?city=rome&genre=trattoria,pizzeria&visited=true`

## 配色（参考）

- 訪問済みピン: 緑系（`#10b981`）
- 未訪問ピン: 赤系（`#ef4444`）
- ジャンルタグ: ジャンル別に淡い色分け（実装エージェントが決定）

## アクセシビリティの配慮

- ピンは色だけでなく **形状やアイコン** でも訪問済み / 未訪問を区別
- タグボタンは `aria-pressed` で状態を提供
- モーダルは `role="dialog"` + フォーカストラップ
