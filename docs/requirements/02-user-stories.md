# 02. ユーザーストーリー

## フォーマット

`<ユーザー>として、<やりたいこと>。なぜなら<理由>から。`

## 旅行者（本人 / 旅行前）

- **US-01**: 旅行者として、訪れる都市ごとに行きたい店をリストアップしたい。なぜなら旅程を組むときに地域ごとの選択肢を一覧で把握したいから。
- **US-02**: 旅行者として、ピッツェリア・トラットリア・ジェラテリアなどジャンル別に店を整理したい。なぜなら「夜はピッツァ、午後はジェラート」と用途で使い分けたいから。
- **US-03**: 旅行者として、店の位置を地図で確認したい。なぜなら宿や観光地からの距離感を視覚的に把握したいから。

## 旅行者（本人 / 旅行中）

- **US-04**: 旅行者として、スマホで現在地の都市に絞り込んで店を見たい。なぜなら今いる場所で行ける店だけを見たいから。
- **US-05**: 旅行者として、店を訪問したら「訪問済み」にマークしたい。なぜなら次回以降のおすすめ整理に使いたいから。
- **US-06**: 旅行者として、気になった店をスマホからすぐ追加したい。なぜなら街歩き中に見つけた店をその場で記録したいから。

## 閲覧者（シェアされた友人）

- **US-07**: 閲覧者として、ログインなしで URL を開いてリストを見たい。なぜならアカウント作成は面倒だから。
- **US-08**: 閲覧者として、自分が行く都市・ジャンルで絞り込みたい。なぜなら自分の旅程に関係する店だけ見たいから。

## ユーザーストーリーと機能の対応

| ストーリー | 対応する機能要件 |
|---|---|
| US-01, US-04, US-08 | F-2（都市絞り込み） |
| US-02, US-08 | F-3（ジャンル絞り込み） |
| US-03 | F-5（地図ピン表示） |
| US-05 | F-7（訪問済みフラグ） |
| US-06 | データソース（Google Sheets）への直接編集 |
| US-07 | 公開・認証不要 |
